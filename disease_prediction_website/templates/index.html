About the Project

What We Built

A complete end-to-end ML pipeline for disease prediction, from raw data to deployed web interface.

📊

Dataset

UCI Heart Disease Dataset — 303 patient records, 13 clinical features including age, cholesterol, blood pressure, and ECG results.

🔬

Data Pipeline

Automated preprocessing: missing value imputation, outlier detection via IQR, feature scaling with StandardScaler, and stratified train/test split.

🤖

ML Models

5 algorithms benchmarked — Logistic Regression, SVM, Decision Tree, Random Forest, and XGBoost with hyperparameter tuning via GridSearchCV.

📈

Evaluation

Comprehensive metrics: Accuracy, Precision, Recall, F1-Score, ROC-AUC, and Confusion Matrix for rigorous model comparison.

💾

Model Saving

Best model persisted with Joblib (.pkl). Includes a reusable prediction script that loads the model and generates confidence scores.

🌐

Web App

Flask-powered real-time prediction interface — enter patient details and get instant disease risk assessment with probability score.

Project Pipeline

ML Workflow

From raw medical data to real-time predictions — a complete end-to-end pipeline.

01
📥
Data Collection
Load UCI Heart Disease dataset. Inspect shape, columns, dtypes and explore distribution of target variable.
→
02
🧹
Preprocessing
Handle missing values, remove duplicates, detect & clip outliers with IQR, encode categoricals, StandardScaler normalization.
→
03
🔍
EDA
Correlation heatmaps, distribution plots, boxplots, count plots. Identify key predictors of heart disease.
→
04
⚙️
Feature Eng.
Select top features via mutual information, drop redundant columns, engineer interaction terms where meaningful.
→
05
🤖
Model Training
Train LR, SVM, DT, RF, XGBoost. Hyperparameter tuning via GridSearchCV. Cross-validation with StratifiedKFold.
→
06
📈
Evaluation
Accuracy, Precision, Recall, F1, ROC-AUC, Confusion Matrix. Best model selected: XGBoost at 90.1%.
